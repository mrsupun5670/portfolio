# Supun Gunasinghe - Portfolio

My personal portfolio website built with React and Three.js.

## About

I'm a fullstack developer and co-founder at ZipZipy Pvt Ltd. This site showcases some of my work in e-commerce, POS systems, and web development.

## Tech Stack

- React + TypeScript
- Three.js for 3D effects
- Tailwind CSS
- Vite

## Projects Featured

- Rasilco Bakers - E-commerce site
- Nayagara LK - Online store
- Redragon Colombo - Gaming peripherals shop
- Various POS applications for retail and restaurants

## Running Locally

```bash
npm install
npm run dev
```
